# Getting Started

From contatooh/:

> `npm install`

> `npm install -g karma-cli`

> `npm install -g grunt-cli@0.1`

> `npm install -g bower && bower install`

> `node server`
or
> `nodemon server`
(npm install nodemon -g)

# Database

> Install MongoDB

> Start `mongod(.exe)`

# Building

>`grunt`

# Testing

Init Karma
>`karma init config/karma.config.js`

Run tests
>`karma start config/karma.config.js`
